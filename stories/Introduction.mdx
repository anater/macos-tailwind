# macOS Tailwind

This project implements the macOS design system using HTML, the Tailwind CSS framework, and vanilla JS. The goal is to adhere to Apple's [Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines) and provide a starting point for developers looking to use the macOS design system to build their web UIs.

The project aims to implement all components in the [macOS 13 Sketch Library](https://developer.apple.com/design/resources/#macos-apps), providing a complete set of components to build apps that feel at home in macOS 13.
